NOTES FOR DEVELOPERS

You need dependencies such as make, g++ and the readline library to build the program

To develop with clangd, please use bear, with the command `bear -- make`
To build, just call the build.sh file on the build folder. For example if you are in the project root folder, do `build/build.sh`

NOTES FOR USERS

To set env vars: `export KEY VALUE`
To unset env vars: `unset KEY`

Alias format on configuration file: `alias=name=command`

There is an example script given in the folder etc, as well as a working configuration
